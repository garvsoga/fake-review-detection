# Fake Review Detection System

AI-based fake review detection using NLP and Machine Learning.

## Features
- Detects fake vs real reviews
- TF-IDF + Logistic Regression
- Flask backend
- Interactive frontend

## How to Run
1. Go to backend
2. Run:
   python clean.py
   python model.py
   python app.py
3. Open frontend/index.html

## Tech Stack
Python, Flask, Scikit-learn, HTML, JS
